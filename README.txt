Instructions on how to make it work:

1. download sublime text 
2. download xampp
3. run xampp-control
4. start apache and mysql
5. paste the folder inside htdocs under xampp directory
6. navigate to localhost/login_sample/index.php


Instructions on how to study the code:

read the comments inside starting from index.php, then signup.php, login.php, 
connection.php, functions.php, and logout.php.
